# Parallel-Programming-Final

## Usage
```bash
$ make run testcase=<testcase_file> algo=<algorithm_type> parallel=<parallel_type>

# Replace <testcase_file> with the path to the testcase file
# Replace <algorithm_type> with the algorithm type (aco or ga)
# Replace <parallel_type> with the parallel type (none, openmp, or thread)
#
# Example:
# make run TESTCASE=./tsp_graph/a280.tsp ALGO=aco PARALLEL=none
```
